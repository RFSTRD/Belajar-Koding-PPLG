Ketentuan Soal
Buatlah desain template web dengan menggunakan CSS (Cascading Style Sheet) seperti ini, dengan ukuran kanvas lebar (width) 700px, dan panjang (height) kurang lebih 700px. Pada kanvas terdiri atas:
  • Header
  • Menu
  • Content
  • Right Sidebar 1
  • Right Sidebar 2
  • dan Footer
Kerjakan langsung di komputer masing masing.
Waktu terbatas: 45 menit (untuk Desain Template CSS ini) & 45 menit untuk Soal Javascript.
