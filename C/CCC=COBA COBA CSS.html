PENGANTAR

Selamat datang di media pembelajaran ini. Di sini, Anda akan mempelajari dasar-dasar pengembangan web.

TUJUAN PEMBELAJARAN

Memahami dasar-dasar HTML, CSS, dan JavaScript untuk membuat halaman web sederhana.

DAFTAR MODUL

1. modul html
2. modul css
3. modul javascript

Modul 1: HTML

HTML adalah bahasa dasar untuk membuat struktur halaman web.

  • Mengenal tag dasar HTML
  • Membuat struktur halaman dengan elemen-elemen HTML

Modul 2: CSS

CSS digunakan untuk mengatur tampilan halaman web.

  • Mengatur warna, font, dan layout
  • Menggunakan selector CSS

Modul 3: JAVASCRIPT

JavaScript adalah bahasa pemrograman untuk memberikan interaktivitas pada halaman web.

  • Membuat interaksi sederhana dengan JavaScript
  • Memanipulasi elemen HTML menggunakan DOM
